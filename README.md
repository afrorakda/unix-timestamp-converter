# #30 Unix Timestamp Converter

A simple, high-contrast tool to convert Unix timestamps (epoch time) into human-readable local, UTC, and ISO date formats.

## Features
- **1 Tool / 1 Action**: Instant conversion from Unix seconds to multiple date formats.
- **Set Current**: One-tap to fetch the current Unix timestamp.
- **High Visibility**: Designed with bold text and clear borders for maximum readability on mobile and desktop.
- **iOS Optimized**: Specific CSS fixes to ensure consistent text rendering on Safari.
- **Privacy First**: Browser-only execution. No tracking, no ads, no login.

## Usage
1. Enter a Unix timestamp (seconds) into the input box.
2. View the converted Local, UTC, and ISO times in the result area.
3. Click "Copy Result" to copy all formats to your clipboard.
